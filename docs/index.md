# NeoOrigins

NeoForge port of Origins with a JSON-first power framework, an event dispatcher, projectile + VFX pipeline, and an extensive pack-author and Java API surface.

## Start here

- **Pack authors** — start with the [Cookbook](COOKBOOK.md), then skim [Pack Format](PACK_FORMAT.md) and the [API Overview](API.md).
- **Java extenders** — read the [Java API](JAVA_API.md) reference and the [Custom Projectiles & VFX](CUSTOM_PROJECTILES.md) guide.
- **Upgrading from 1.x** — follow the [Migration Guide](MIGRATION.md) and skim [What's New in 2.0](2_0_CHANGES.md).
